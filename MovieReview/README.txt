Основной код приложения находится в папке app.

Запуск приложения:

1. Запустить run.py
2. Дождаться, когда будет готов классификатор. (5-6 сек.)
3. Открыть браузер
4. В браузере ввести http://localhost:5000
5. Окажемся на странице Home. Перейти на страницу Analyzer для использования анализатора.
